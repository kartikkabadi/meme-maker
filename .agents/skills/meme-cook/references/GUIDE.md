# Meme Cook — Template & Caption Guide

## Caption rules

- 1-3 short lines per text slot.
- Use the thread's own language, quotes, or slang.
- Avoid long paragraphs and overstuffed captions.
- Match the template's visual logic; the image should reinforce the joke, not just carry text.
- One strong line is usually better than three OK lines.

## How to pick a good template

- The template should feel like it was made for the joke.
- Prefer concise, recognizable formats with 1-4 text slots.
- If the joke is about escalation, use an escalation template.
- If the joke is about a reveal or realization, use a reveal template.
- If the joke is about a false promise, use a deception template.

## Verifying a render

- Check that text is centered/fits inside its slot.
- Check that the font, stroke, and contrast are readable.
- Check that the caption does not cover the most important part of the image.
