## 1. Anthropic MCP community (#show-and-tell)

> Built an MCP server that gives your agent memes: meme-maker — headless, deterministic meme generation. 609 templates (images + GIFs), one JSON spec in, pixels out; `render_meme` returns the image inline. Runs fully local — no cloud, no accounts, no telemetry. Install: `curl -fsSL https://raw.githubusercontent.com/kartikkabadi/meme-maker/main/install.sh | sh`, then add `meme-maker-mcp` to your `mcpServers`. Curious what MCP hosts people want tested — repo: https://github.com/kartikkabadi/meme-maker

## 2. LangChain Discord (#show-your-work)

> Shipped meme-maker: a meme engine for AI agents. Works as a CLI with `--json`, an HTTP API, or an MCP server, all local (MIT, no cloud).
