# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |
| < 0.3   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, use [GitHub private vulnerability reporting](https://github.com/kartikkabadi/meme-maker/security/advisories/new)
to submit a report. Include a description of the issue, steps to reproduce,
and the affected version.

You can expect an acknowledgement within 7 days. Once the issue is confirmed,
a fix will be prepared and released, and you will be credited in the release
notes unless you prefer otherwise.

## Scope

This project renders images from untrusted text input via Sharp and SVG.
Reports of path traversal, SVG/XML injection, resource-exhaustion, or
dependency vulnerabilities are especially welcome.
